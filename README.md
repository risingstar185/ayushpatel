## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/risingstar185/ayushpatel.git
```

Go to project directory:

```
cd ayushpatel
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```
